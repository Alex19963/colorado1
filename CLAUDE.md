# CLAUDE.md
## Контекст проекта для Claude Code

### 📌 Название проекта
**Colorado** — веб-приложение для учета рабочего времени сотрудников, управления заданиями и расчета заработной платы.

---

## 🎯 Цель проекта
Создание современного и быстрого веб-приложения с тёмной темой, интуитивно понятным интерфейсом и стабильной backend-архитектурой.

---

## 🛠️ Используемые технологии

### Frontend
- HTML5
- CSS3
- Vanilla JavaScript
- Адаптивный дизайн
- Тёмная тема

### Backend
- Node.js
- Express.js
- SQLite3
- Server-Sent Events (SSE) для обновлений в реальном времени

### Зависимости
- cors
- express
- sqlite3
- uuid
- bcryptjs

---

## 📁 Структура проекта

- Не изменять дизайн-систему без необходимости.
- Соблюдать тёмную тему интерфейса.
- Использовать только Vanilla JavaScript.
- Не добавлять новые библиотеки без согласования.
- Все ошибки выводить в интерфейсе, а не через alert().
- Сохранять структуру базы данных SQLite.
- Поддерживать единый стиль модальных окон и таблиц.
- Не изменять существующие API без необходимости.
- Писать чистый и оптимизированный код.
